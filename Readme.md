\# LangSwitch



Система переключения раскладки Windows через демон на C# и горячие клавиши AutoHotkey.



\## Состав



\- \*\*daemon/\*\* — C# демон, который принимает команды через Named Pipe.

\- \*\*ahk/\*\* — два скрипта:

&#x20; - AltSwitch.ahk — переключение по RAlt

&#x20; - CtrlSwitch.ahk — переключение по RCtrl

\- \*\*installer/\*\* — инсталлятор (Inno Setup)

\- \*\*docs/\*\* — документация



\## Команды демона



\- `toggle\_alt` — переключение по логике RAlt

\- `toggle\_ctrl` — переключение по логике RCtrl

\- `log\_on` — включить логирование

\- `log\_off` — выключить логирование



\## Автозапуск



Демон и AHK‑скрипт добавляются в автозагрузку через инсталлятор.
